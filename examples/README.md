# examples/ — Ілюстративні вибірки

Ці файли містять по 10–20 записів кожного типу для демонстрації формату і структури даних. Вони **не** є частиною повного корпусу і не дозволяють його відтворити.

## Файли

| Файл | Зміст | Записів |
|------|-------|---------|
| `extract_sample.jsonl` | Вивід `extract.py` — формат `{id, version, text}` | ~15 |
| `aligned_sample.jsonl` | Вивід `align.py` — формат `{id, version, en, uk_human}` | ~15 |
| `classified_sample.jsonl` | Вивід `classify.py` — з `metadata.type` і `uk_human_quality` | ~20 (по 3 на тип) |
| `gemini_results_sample.jsonl` | Вивід `gemini.py` — з `uk_gemini` | ~15 |

Тексти у цих файлах є власністю Larian Studios. Вони включені виключно для ілюстрації формату даних у межах академічного fair use. Не перевидавати і не розповсюджувати окремо від наукового контексту.
